Консольное приложение для шифрования и расшифровки текстовых сообщений с использованием **классического метода Виженера**.

Программа позволяет работать как с одним ключом, так и с несколькими ключами последовательно.

---

## 📌 Описание проекта

«Шифровальщик сообщений» — это приложение, реализующее симметричное шифрование по методу Виженера.  

Пользователь может:

- 🔒 Зашифровать сообщение и сохранить результат в файл  
- 🔓 Расшифровать сообщение из файла  
- 🔑 Использовать один или несколько ключей  

---

## 🧠 Метод Виженера

Метод Виженера — это полиалфавитный шифр, в котором каждая буква исходного текста сдвигается на количество позиций, определяемое соответствующей буквой ключа.
